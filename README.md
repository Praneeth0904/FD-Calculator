## 📌 FD Calculator

Calculates the amount for a fixed deposit by the user's input.

---

## 🛠️ Tech Stack

HTML5 • CSS3 • JavaScript

---

## ✨ Features

* Calculates the fixed deposit amount by providing principal amount, interest, and time.
* Provides an easy and engaging user experience.
